# Excel Shortcuts – Master Reference

This file contains all important Excel shortcuts collected throughout the learning journey.

---

## Formatting
- Ctrl + 1 → Format Cells
- Ctrl + B → Bold
- Ctrl + Shift + $ → Currency Format
- Ctrl + Shift + % → Percentage Format

---

## Filling & Patterns
- Ctrl + D → Fill Down
- Ctrl + R → Fill Right
- Ctrl + E → Flash Fill

---

## Sorting & Filtering
- Alt + A + S + A → Sort A to Z
- Alt + A + S + D → Sort Z to A
- Alt + A + T → Toggle Filter

---

## Freeze Panes
- Alt + W + F + F → Freeze Panes

---

## Find & Replace
| Shortcut  | What it does   |
|------------|------------|
| Ctrl + F | Find |
| Ctrl + H | Replace |
| Ctrl + G | Go To |

---

## Basic Formula & Navigation Shortcuts
| Shortcut               | What It Does                          | Why It Matters                  |
| ---------------------- | ------------------------------------- | ------------------------------- |
| `=`                    | Start any formula                     | Fast formula entry              |
| `Alt + =`              | AutoSum                               | Quickly apply SUM               |
| `Ctrl + Enter`         | Fill selected cells with same formula | Apply formula to multiple cells |
| `Ctrl + D`             | Fill Down                             | Copy formula downward           |
| `Ctrl + R`             | Fill Right                            | Copy formula to right           |
| `F2`                   | Edit selected cell                    | Quickly modify formula          |
| `F4`                   | Lock cell reference ($)               | Convert A1 → $A$1               |
| `Ctrl + Arrow Keys`    | Jump to end of data                   | Fast navigation                 |
| `Ctrl + Shift + Arrow` | Select entire data range              | Fast selection                  |
| `Ctrl + Shift + L`     | Apply/Remove filters                  | Data filtering                  |
